ЧЕСТНОСТЬ ИГРЫ И PROVABLY FAIR

InOut Games использует криптографическую систему верификации. Каждый результат проверяем и неподделен.

🔐

SERVER SEED

Сервер генерирует зашифрованный seed-хэш до начала раунда и публикует его хэш. Это гарантирует, что результат не может быть изменён после начала игры.

🎲

CLIENT SEED

Ваш браузер добавляет собственный client seed к генерации. Это означает, что ни одна из сторон не может контролировать финальный результат единолично.

✅

ВЕРИФИКАЦИЯ

После раунда InOut Games раскрывает server seed. Вы можете самостоятельно проверить результат, применив SHA-256 хэш-функцию к комбинации seeds.

КАК ПРОВЕРИТЬ РЕЗУЛЬТАТ

1

Скопируйте Server Seed Hash до раунда

2

Сыграйте раунд, запомните Client Seed

3

Получите раскрытый Server Seed после раунда

✓

Проверьте SHA-256(server+client) = хэш
